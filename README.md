## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/toolverse.git
```

2. Navigate to the project directory:
```bash
cd toolverse
```

3. Open `index.html` in your web browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```
